🌍 Africana Ultimate Solutions Website

A modern, responsive website built for Africana Ultimate Solutions, an education consulting brand focused on empowering educators, institutions, and organizations through innovation and technology.

⸻

✨ Overview

This project is a fully responsive website designed to showcase the services, vision, and impact of Africana Ultimate Solutions.

It serves as:
	•	A digital presence for the brand
	•	A lead generation platform (consultations, inquiries)
	•	A content hub for insights and resources

⸻

🚀 Features
	•	📱 Responsive Design (mobile-first)
	•	🎯 Clear call-to-action sections (Book Consultation, Get Started)
	•	🧭 Smooth navigation with scroll effects
	•	🎨 Clean and modern UI/UX
	•	📰 Blog section for thought leadership
	•	📩 Contact form for client inquiries

⸻

🛠️ Tech Stack
	•	HTML5 – Structure
	•	CSS3 – Styling & layout
	•	JavaScript – Interactivity
	•	Smooth Scroll JS – Navigation enhancement
